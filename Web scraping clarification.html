Scraping Instagram, especially for large volumes like 1 lakh posts or comments on the topic of IPL, is tricky because:

  • Instagram's API is limited and mostly private now.

  • They have strong anti-scraping measures.

  • Automated scraping of Instagram often violates their Terms of Service.

  • Getting 100,000 posts on a specific hashtag/topic is huge and may require advanced tools or paid services.

Here’s a practical, ethical approach you can try:

1. Use the Instagram Hashtag Search for #IPL

Instagram shows public posts tagged with #IPL. You can collect data like captions, usernames, and timestamps from public posts.

2. Tools & Libraries:

  • Instaloader — a Python tool to download Instagram posts by hashtags, profiles, etc.

  • Selenium — to automate browser scrolling to load more posts.

  • Save data to Excel with pandas.

Sample Python snippet using Instaloader to get posts from #IPL:

python
import instaloader
import pandas as pd

L = instaloader.Instaloader()

# Hashtag to scrape
hashtag = 'IPL'

posts = instaloader.Hashtag.from_name(L.context, hashtag).get_posts()

data = []

for i, post in enumerate(posts):
    if i >= 1000:  # Limit to 1000 posts for demo (you can increase but careful about limits)
        break
    data.append({
        'username': post.owner_username,
        'caption': post.caption,
        'date': post.date.strftime('%Y-%m-%d %H:%M:%S'),
        'likes': post.likes,
        'comments': post.comments,
        'url': post.url
    })

df = pd.DataFrame(data)
df.to_excel('ipl_instagram_posts.xlsx', index=False)

print("Data saved to ipl_instagram_posts.xlsx")

Important notes:

  • Instaloader respects Instagram's limits, but scraping huge volumes may cause your IP/account to be blocked.

Twitter is way more scrape-friendly (relatively) because of its official APIs and better public access. If you want to collect lots of tweets about IPL (Indian Premier League), here’s how you can do it effectively:

Recommended approach: Use Twitter API v2

  • Twitter’s API lets you search tweets by keywords, hashtags, date ranges, etc.

  • You can collect tweet text, username, date, likes, retweets, etc.

  • Then export the data to Excel.

How to get started:

  1. Create a Twitter Developer Account
    Go to developer.twitter.com, apply for a developer account, and create a project/app.
    You’ll get API keys and Bearer Token.

  2. Use Python Tweepy Library
    Tweepy supports Twitter API v2 and makes it easy to collect tweets.

Sample Python code to scrape tweets about IPL and save to Excel

python
import tweepy
import pandas as pd

# Your Twitter API credentials
bearer_token = 'YOUR_BEARER_TOKEN_HERE'

# Authenticate client
client = tweepy.Client(bearer_token=bearer_token)

# Query: search tweets containing 'IPL'
query = 'IPL -is:retweet lang:en'  # Exclude retweets, English only

tweets_data = []

# Twitter API limits max 100 tweets per request
# Use pagination to get more tweets

paginator = tweepy.Paginator(client.search_recent_tweets,
                             query=query,
                             tweet_fields=['created_at', 'public_metrics', 'text', 'author_id'],
                             max_results=100)

max_tweets = 10000  # Number of tweets you want to collect
count = 0

for response in paginator:
    for tweet in response.data:
        tweets_data.append({
            'tweet_id': tweet.id,
            'author_id': tweet.author_id,
            'text': tweet.text,
            'created_at': tweet.created_at,
            'retweet_count': tweet.public_metrics['retweet_count'],
            'reply_count': tweet.public_metrics['reply_count'],
            'like_count': tweet.public_metrics['like_count'],
            'quote_count': tweet.public_metrics['quote_count']
        })
        count += 1
        if count >= max_tweets:
            break
    if count >= max_tweets:
        break

# Convert to DataFrame and save as Excel
df = pd.DataFrame(tweets_data)
df.to_excel('ipl_tweets.xlsx', index=False)

print(f"Collected {count} tweets and saved to ipl_tweets.xlsx")

Important notes:

  • search_recent_tweets gives tweets from last 7 days only (free tier limit). For older tweets, you need Academic or Elevated access.

  • API rate limits apply — you may need to handle delays or pauses.

  • You can filter language, exclude retweets, search for hashtags (#IPL), or add date ranges.

What is a Twitter Bearer Token?

A Bearer Token is a special type of access token that Twitter uses to authenticate your API requests when you want to access Twitter’s data programmatically.

  • It’s part of the OAuth 2.0 authentication system.

  • When you use the Bearer Token, you don’t need to send username/password for each request.

  • It tells Twitter “This app is authorized to access data”.

  • Required to make requests to Twitter API v2 (the modern Twitter API).

How to get your Twitter Bearer Token?

  1. Sign up for a Twitter Developer account:
    Go to https://developer.twitter.com/ and apply for a developer account if you don’t have one.

  2. Create a Project and App:
    After approval, create a new project and app inside the Twitter Developer Portal.

  3. Get API Keys and Tokens:

    • Go to your App’s “Keys and Tokens” section.

    • You will see API Key & Secret and Bearer Token.

    • Copy the Bearer Token — this is what you use in your Python scripts to authenticate requests.

Summary:

  • The Bearer Token is like a password your code uses to prove it’s authorized.

  • Never share your Bearer Token publicly — treat it like a secret password.
